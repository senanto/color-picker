# Color Picker

A lightweight, browser-based image color picker. Extract HEX, RGB, HSV, and CMYK values from any image instantly.

## Features

- **Multiple upload methods** — Drag & drop, file picker, Ctrl+V paste, or image URL
- **Live magnifier** — Hover to see a zoomed-in pixel grid with crosshair
- **Four color formats** — HEX, RGB, HSV, and CMYK displayed simultaneously
- **Click to save** — Pick colors and build a palette in real time
- **One-click copy** — Copy any color value to clipboard instantly
- **Privacy-first** — All processing happens in the browser, no server uploads

## Usage

1. Open `index.html` in any browser
2. Upload an image via drag & drop, file picker, Ctrl+V, or URL
3. Hover over the image to preview colors with the magnifier
4. Click to pick and save a color
5. Copy any value with the 📋 button
